一款参照《艾希》，由三个人制作的游戏，作为我们的程序设计范式的期末项目。

基础功能

1. 角色系统
(1)支持主角模型——艾希
(2)支持状态切换（待机-奔跑-普攻-释放技能-冲刺-跳跃-受击后退）
(3)支持某些基本动作叠加（跳跃的同时冲刺）
(4)支持基本动作：待机；奔跑；跳跃（支持多段连跳）；冲刺；受到攻击后退；普通攻击；释放两种技能

2. 战斗系统
(1)玩家生命值
(2)玩家护盾值
(3)敌人生命值
(4)受击硬直
(5)战斗判定：实时接触检测；攻击范围判定；追击范围判定

3. 敌人与 Boss
(1)两种近战普通怪，一个带盾精英怪，一个远程攻击敌人，三个BOSS
(2)敌人具备基本AI：追击玩家；近战和远程两种攻击模式；在游戏场景内巡逻

4. 场景功能
(1)场景动态音乐，一个场景一首BGM，音乐随场景切换而切换
(2)完备的场景切换逻辑
(3)2D滚动长卷轴式地图

5. UI功能
(1)标题菜单
(2)玩家血条和护盾条
(3)死亡重开
(4)暂停/继续菜单
(5)物品背包（显示当前已经收集到的物品，显示物品详细信息）
(6)成就栏（显示已获得的成就和达成条件）
(7)伤害显示（玩家角色攻击后会显示伤害值）

6. 音频系统
(1)游戏BGM（随游戏场景切换）
(2)旁白音频
(3)战斗音效（普攻音效，技能音效，受击音效）
(4)以上三种声音的完备播放逻辑

扩展功能

1.旁白系统（META元素）
2.成就系统
3.物品收集系统
